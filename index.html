Quality dimensions¶

We have identified three (3) data quality dimensions problems : Data Consistency with the "Subject's age" and "Imputation probability" variables, Data Accuracy with the "Full Address" and "Location of death (state)" variables and Data Completeness with the "Subject's race" variable and many others. We have looked for other data quality dimensions such as Data Uniqueness too but there are no duplicated persons in the dataset.

Methodology¶

In this section, we are going to indicate the quality goals, questions, related quality factors and metrics.

Data consistency with "Subject's age"¶

Goal : Standardize the age so that all the values have consistency
Question : Which kind of formats are present in this variable ?
Quality factor : Domain integrity
Metric : Ratio of ages that are not in the expected format

Quality Plan :

  • Detect the ages that are not in the correct format
  • Define a methodology to correct most of the values which are in the wrong format

Data Accuracy with "Location of death (state)"¶

Goal : Replace all the abbreviations with the real values
Question : Which kind of formats are present in this variable ?
Quality factor : Precision
Metric : Average length of the values

Quality Plan :

  • Use a dataset which have both the real values and the abbreviation
  • Join this dataset with ours and replace the states with their real values

Data Completeness with "Subject's race"¶

Goal : Replace the null values (Race unspecified)
Question : Which kind of formats are present in this variable ?
Quality factor : Density
Metric : Ratio of null values

Quality Plan :

  • Use the "Imputation probability" variable with a threshold
  • If the condition is satisfied, replace the null value with the "Subject's race with imputation" value
